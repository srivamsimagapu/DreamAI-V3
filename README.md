# 🌙 DreamAI v3

## Overview

DreamAI v3 is an AI-powered dream journal designed to help users record, analyze, visualize, and explore their dreams. The project combines Python, AI, NLP, speech processing, and image-generation concepts into a single platform.

---

## Features

✅ Dream Recording

* Save dream titles and descriptions
* Store dreams in a local database
* Browse dream history

✅ AI Analysis

* Dream mood detection
* Character extraction
* Recurring dream detection
* Dream categorization

✅ Speech Features

* Speech-to-text dream recording
* Text-to-speech playback

✅ Image Generation Ready

* Convert dreams into image prompts
* Integration-ready for Stable Diffusion and other AI image models

✅ Search & Analytics

* Dream similarity search
* Basic analytics dashboard
* Dream trend tracking
